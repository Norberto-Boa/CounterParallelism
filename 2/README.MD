### Installing

You must navigate to the Assignment directory and then install all the dependencies needed for this project.

Navigate to the project directory

```
cd Assignment/2/
```

Install all the dependencies needed for this project

```
npm install 
```

Run the project

```
node Exercise2.js 
```
